# 数据库规范

> 面向项目的数据库相关规则，聚焦命名、SQL、事务、索引、数据一致性与变更管理。

---

## 概览

数据库规范解决的是“数据是否可读、SQL 是否可维护、事务是否稳定、变更是否可控”。

很多项目代码表面看着整齐，数据库层却写得像野生摊位：命名混乱、`select *` 满天飞、事务边界飘忽、索引随缘。真出事故时，锅最后都得数据库背，但代码的人也别装无辜。
